## IV. PARALELNA AKCIJA JE LEGITIMNA PRAKSA

### A. Međunarodno Prihvaćena Praksa

#### ✅ ICIJ / OCCRP Model (Global Investigations)

```
KADA ISTRAŽUJU VELIKI SLUČAJEVI:

1. Novinari → Istraživanja (1-2 godine)
2. Paralelno: Kontakt sa tužilacom
3. Istovremeno: Objavljivanje + Prijava

REZULTAT: Panama Papers (2016)
- 400 novinara
- 100 medija
- Objavljivanje + Istrage = Simultano
→ LEGALNO JE
→ EFIKASNO JE
```

#### ✅ EU Antifraud (OLAF) Model

```
OLAF PROCEDURA:

Kada baci sumnja:
1. Primljaću prijave od javnosti
2. Paralelno sa privatnom istragom
3. Objavljivanjem kroz medije
4. Svetla reflektora = Zaštita

LEGITIMNO JE jer:
- Javna akcija je pravo
- Mediski izveštaji su sloboda
- Paralelnost je zakonita
```

#### ✅ DRI (Srbija) Model

```
DRI PRAKSA:

Kada revidiram:
1. Čuju od javnosti / medija
2. Pokrenuti revizije
3. Izveštaji su JAVNI (objavljeni)
4. Mediji pisali = PARALELNA akcija

DRI JE KORISTIO OVO - sveće me može biti sporno!
```
